The Challenge

Our task was to design, build, and develop an interactive game for mobile or desktop play, with the theme of "Home" - interpreted as we saw fit. I proposed the initial game concept of "Get Home", which uses "home" both to represent the home screen of the game, as well as "home" in the more personal sense. "Get Home" gamifies a relatable experience many of us have had growing up with varying degrees of access to technology. The premise of this desktop game is simple - you are a teenager, messing around on your computer, when you hear your mother approaching your room. Your goal is to clear the non-homework-related applications and windows from your computer screen, before the timer goes off, representing Mom's arrival.

I was the lead visual designer on the game development team, and created reusable game elements such as application windows, desktop templates, and application icons.

The Goal

Our goal was to create several reusable game images and templates that could be inserted into the game interface, allowing the users to interact with varying game elements. These reusable elements include application icons, windows of different dimensions, and window content.

As a final product, our goal was to create an interactive game that can be played like such: the user is presented with a desktop filled with various application windows, and a timer in the corner. The user's goal is to close all application windows, which sometimes re-spawn, before the timer goes off.

Approach

We used Unity as the primary tool for game development. I, as the UI designer, used Adobe XD to design reusable UI elements. We also used Adobe Photoshop for illustrated game elements.

The Outcome

We presented our game as part of Global Game Jam 2019. It is available for download here.
